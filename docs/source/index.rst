MatlabHelper documentation
==========================

MatlabHelper is a PyQt6 desktop application that streams the contents of a selected
``.m`` file into the currently focused MATLAB editor window. Use this documentation to
explore the architecture, services, and user interface guidelines that empower the
teaching workflow.
